Framer CSS

Framer is a minimalist Sass boilerplate that provides helpful, browser-consistent styling for default HTML elements - buttons, forms, tables, lists, and typography.

Containers

There are three container sizes for content that are centered with left and right padding. Content without a container will span the width of the viewport.

Grid

Simple flex grid system with built-in gutter. For sizes, .half, .one-fourth, .three-fourths, .one-third, and .two-thirds classes are available.

Forms

All inputs, select, and buttons are normalized for a common height cross-browser so inputs can be stacked or placed alongside each other. 


<form>
<div class="o-flex margin-bottom">
  <div class="o-flex__item o-flex__item--half">
    <label for="name">Your email</label>
    <input
      type="email"
      id="email"
      placeholder="Enter your email here.."
    />
  </div>
  <div class="o-flex__item o-flex__item--half">
    <label for="choose">Reason for contact</label>
    <select id="choose">
      <option disabled selected>Please select</option>
      <option value="option-1">Option 1</option>
      <option value="option-2">Option 2</option>
    </select>
  </div>
</div>
<div class="o-flex margin-bottom">
  <div class="o-flex__item">
    <label for="message">Message</label>
    <textarea id="message"></textarea>
  </div>
</div>
<div class="o-flex">
  <button class="c-button c-button--primary" type="submit">
    Submit
  </button>
  <label>
    <input type="checkbox" />
    Remember me
  </label>
</div>
</form>

From validation

For a successful action you can use the c-forms--success class. If there are errors, you may use c-forms--error.
